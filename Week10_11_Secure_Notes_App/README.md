## 📌 Overview
Secure Notes is a Flutter-based mobile application designed to securely store personal notes.

---

## 🚀 Features
- Biometric Authentication
- PIN Login
- Create/Edit/Delete Notes
- Search Notes
- Auto Lock
- Logout
